# bluetooth-api
Experiment with Arduino, Bluetooth module and Bluetooth API.

## Demo

## What do you need
* Arduino
* Bluetooth module v4 (HM-10 in my case)
* nRF Connect app (in my case)
* Led

## How run it
* Upload sketch (ble2.ino) to your Arduino

* Connect your module and led to Arduino
  * Module
  * GND to GND
